# Changelog

## v1.0.0 — manuscript submission snapshot

- Uses the 198 original Collar-level observations without smoothing,
  interpolation, outlier replacement, or group-mean reconstruction.
- Constructs 132 within-stratum treatment/CK log ratios.
- Estimates four pre-specified treatment × monitoring-window effects.
- Uses CSU-clustered robust covariance for the primary analysis.
- Adds randomized-block-clustered sensitivity estimates.
- Adds linear contrasts for October-minus-July treatment effects.
- Adds descriptive CSU-level window-specific and across-window summaries.
- Generates Figure 1, Figure 2, and Supplementary Figure S1.
